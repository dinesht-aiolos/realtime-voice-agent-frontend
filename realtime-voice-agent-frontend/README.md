# 🎙️ Realtime Voice Agent Frontend

This is the **React.js frontend** for the Realtime Deepgram Voice Agent.  
It allows users to **speak via microphone**, sends audio to the backend, and receives **agent responses in real-time**.

---

## 🚀 Features

- Real-time microphone capture
- WebSocket streaming to backend
- Display user messages and agent responses
- Play audio responses from Deepgram
- Easy integration with your Python backend

---

## 📦 Tech Stack

- React.js
- WebSockets
- Browser MediaRecorder API
- CSS for styling
